🛍️ Shopping Behaviour Analytics Dashboard

A complete end-to-end analytics project that uncovers customer shopping patterns using **Power BI**, **Python**, and **Grok API**. This dashboard provides deep insights into sales trends, customer behaviour, category performance, promotional impact, and payment method profitability.

📌 Project Overview

This project analyzes customer shopping behavior using a retail dataset sourced from **Kaggle**.
The goal is to understand:

Buying trends across months, age groups, and seasons
Top-selling product categories
Impact of promotional discounts
Customer spending habits
Profit contribution from different payment methods

All insights are visualized in an interactive **Power BI Dashboard

📊 Key Features

📈 KPI Cards: Total Sales, Total Customers, Orders, Avg Rating, Promo %, Profit Margin
🕒 Trend Analysis: Month-wise sales and orders
🛒 Category Performance: Promo sales vs customer spend
🎯 Top 10 Items: Best-selling products
💳 Payment Method Insights:Orders & profit comparison
👥 Customer Segmentation: Age group, season, and year filters

🛠️Tools & Technologies Used
Power BI – For data visualization & dashboard creation
Python– For preprocessing, cleaning, and analysis
Excel– For initial dataset handling
Kaggle Dataset– Retail shopping behaviour data

📁 Dataset

The dataset is sourced from Kaggle, containing customer demographics, purchase records, ratings, order details, payment modes, and season-based buying patterns.
